# لوحة تحليل شقق الرياض 🏢

تطبيق Streamlit احترافي لتحليل بيانات شقق الرياض مع رسوم بيانية تفاعلية.

## المميزات 📌

- **لوحة تحكم تفاعلية** مع فلاتر متقدمة
- **6 رسوم بيانية** مختلفة لتحليل البيانات
- **معلومات إحصائية** شاملة
- **جدول بيانات** يمكن الفرز والتصفية عليه
- **واجهة عربية كاملة** 100% (RTL)

## الرسوم البيانية المتضمنة 📊

1. **توزيع الأسعار** - Histogram لتوضيح توزيع أسعار الشقق
2. **العلاقة بين المساحة والسعر** - Scatter Plot ملون حسب عدد الغرف
3. **عدد الشقق حسب المنطقة** - Bar Chart أفقي
4. **متوسط السعر حسب الغرف** - Bar Chart عمودي
5. **تأثير المصعد على السعر** - مقارنة بين الشقق بمصعد وبدونه
6. **توزيع الشقق حسب الغرف** - Pie Chart

## الفلاتر المتاحة 🔍

- **اختيار المنطقة** - متعدد الخيارات
- **نطاق السعر** - slider للاختيار بين الحد الأدنى والأقصى
- **عدد الغرف** - متعدد الخيارات
- **وجود المصعد** - Yes/No

## متطلبات التشغيل 💻

- Python 3.8+
- المكتبات المذكورة في `requirements.txt`

## طريقة التثبيت والتشغيل 🚀

### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 2. تشغيل التطبيق
```bash
streamlit run riyadh_apartments_analysis.py
```

### 3. فتح المتصفح
التطبيق سيفتح تلقائياً على `http://localhost:8501`

## هيكل الملفات 📁

```
.
├── riyadh_apartments_analysis.py     # كود التطبيق الرئيسي
├── riyadh_apartments_data.csv         # البيانات
├── requirements.txt                    # المكتبات المطلوبة
└── README.md                           # هذا الملف
```

## نصائح الاستخدام 💡

- استخدم الفلاتر في الشريط الجانبي لتصفية البيانات
- مرر الماوس على الرسوم البيانية لرؤية التفاصيل
- انقر على المفاتيح في الرسوم البيانية لإظهار/إخفاء البيانات
- الجدول في الأسفل يعرض جميع البيانات المفلترة الحالية

## الإحصائيات المتاحة 📈

يعرض التطبيق:
- إحصائيات السعر (الحد الأدنى، الحد الأقصى، الوسيط، الانحراف المعياري)
- إحصائيات المساحة بنفس المعايير
- نسبة الشقق المؤثثة
- نسبة الشقق ذات المصعد
- متوسط عمر العقار

## ملاحظات تقنية ⚙️

- البيانات يتم تحميلها مرة واحدة فقط (caching)
- جميع الرسوم البيانية تفاعلية (interactive charts)
- الواجهة responsive وتعمل على الهواتف الذكية
- تم استخدام Plotly للرسوم البيانية عالية الجودة
